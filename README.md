# kettle-project

## 第一步：安装项目依赖
```
yarn install or npm run install
```

### 第二步：运行项目
```
yarn run dev:mp-weixin or npm run dev:mp-weixin
```

### 第三步：dist文件夹下找到mp-weixin文件，微信开发者工具打开
```
/dist/dev/mp-weixin
```

### 如何上传体验版：
```
微信开发者工具打开mp-weixin项目，点击上传即可得到体验版
```

### 如何发布正式版：
```
微信小程序管理后台选择一个体验版进行审核，审核通过后即可提交上线
```
